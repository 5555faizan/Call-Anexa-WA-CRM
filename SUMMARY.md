### Quick Start (Easiest)
```bash

```

### Manual Start
```bash
# Terminal 1 - Backend
cd backend
npm start

# Terminal 2 - Frontend  
cd frontend
npm run dev
```

### Then Open
```
http://localhost:5173
```

## 🐛 Common Issues & Fixes

**Issue:** Dependencies not installed
**Fix:** `npm install` in both backend & frontend

**Issue:** Port already in use
**Fix:** Close other apps using port 3001/5173

**Issue:** QR not showing
**Fix:** Refresh page, check backend running

**Issue:** AI not responding
**Fix:** Check API key, enable AI, verify balance
