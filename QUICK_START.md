# 🚀 Quick Start Guide

## Installation (5 Minutes!)

### Step 1: Install Dependencies

**Backend:**
```bash
cd backend
npm install
```

**Frontend:**
```bash
cd frontend
npm install
```

### Step 2: Start Servers

**Option A: Automatic (Windows)**
Double click `start.bat` file

**Option B: Manual**

Terminal 1 (Backend):
```bash
cd backend
npm start
```

Terminal 2 (Frontend):
```bash
cd frontend
npm run dev
```

### Step 3: Open Browser
Open: `http://localhost:5173`

### Step 4: Create Account
- Click "Register"
- Enter username & password
- Login

### Step 5: Create Agent
- Click "New Agent"
- Name your agent
- Scan QR with WhatsApp
- Done! 🎉

---

## Features Usage

### 🤖 Enable AI Responses
1. Click "AI" button on agent
2. Toggle "Enable AI"
3. Select provider (OpenAI/Gemini)
4. Enter API key
5. Save

### 📊 View Analytics
- Click "Stats" button
- See all message statistics

### ⏰ Schedule Messages
1. Click "Schedule" button
2. Enter contact (format: 923001234567@s.whatsapp.net)
3. Write message
4. Select date & time
5. Schedule!

## API Keys

### OpenAI
- Get from: https://platform.openai.com/api-keys
- Cost: ~$0.002 per 1000 messages (GPT-3.5)

### Gemini (Recommended for Testing)
- Get from: https://ai.google.dev
- Free tier available!

## Troubleshooting

**QR not showing?**
- Refresh page
- Check backend is running
- Check console for errors

**Messages not sending?**
- Check agent is "Active" (green)
- Verify WhatsApp is connected
- Check reply delay setting

**AI not working?**
- Verify API key is correct
- Check API balance/quota
- Enable AI in settings
